# Обозреватель BTC RPC

[![версия npm][npm-ver-img]][npm-ver-url] [![загрузки NPM][npm-dl-img]][npm-dl-url]


Простой обозреватель биткойн-блокчейнов без базы данных, работающий через RPC-интерфейс. Создан с помощью Node.js, express, bootstrap-v4.

Этот инструмент задуман как простой самостоятельный обозреватель блокчейна Биткоина, использующий RPC-команды, отправляемы вашему узлу bitcoind. Этот инструмент прост в использовании, но в настоящее время ему не хватает функций обозревателей на основе базы данных.

Я создал этот инструмент, потому что хотел использовать его сам. Какими бы ни были причины запуска полного узла (ненадежность, технический интерес, поддержка сети и т. д.), полезно оценить «полноту» вашего узла. С помощью этого обозревателя вы можете не только исследовать блокчейн (в традиционном понимании термина «обозреватель»), но и исследовать функциональные возможности собственной ноды.

Демонстрационные обозреватели доступны по адресу:

* BTC: https://btc.chaintools.io  ?
* LTC: https://ltc.chaintools.io  ?

* SKYR: http://explorer.skynet-coin.com

# Функции

* Просмотр блоков
* Просмотр сведений о блоке
* Просмотр сведений о транзакциях с навигацией «назад» по проведенным транзакциям
* Просмотр содержимого JSON, используемого для создания большинства страниц
* Поиск по идентификатору транзакции, хешу/высоте блока и адресу
* Необязательная история транзакций для адресов, основанная на запросах к настроенным серверам ElectrumX.
* Сводка мемпула с разбивкой по комиссиям, размеру и возрасту
* Браузер команд RPC и терминал
* В настоящее время поддерживают BTC, LTC, SKYR (можно легко добавить поддержку любой монеты, совместимой с протоколом Bitcoin-RPC)

# Вступление

Приведенные ниже инструкции ориентированы на BTC, но их можно легко адаптировать и к другим монетам.

## Подготовка

1. Установите и запустите полноценную архивирующую ноду - [инструкции](https://bitcoin.org/en/full-node). Убедитесь, что на вашем биткойн-узле включена полная индексация транзакций (`txindex=1`) и включен сервер RPC (`server=1`).
2. Синхронизируйте свой узел с сетью Биткойн.
3. Установите «последнюю» версию Node.js (рекомендуется 8+).

## Инструкции

```bash
npm install -g skyr-rpc-explorer
skyr-rpc-explorer
```

Если вы работаете в основной сети с каталогом данных и портом по умолчанию, это должно сработать.
Откройте [http://127.0.0.1:3002/](http://127.0.0.1:3002/) для просмотра проводника.

Вы можете установить параметры конфигурации в файле `.env` или с помощью аргументов CLI.
Подробнее см. [configuration](#Конфигурация).

### Конфигурация

Параметры конфигурации могут быть переданы как переменные среды
или заданы в файле env в `~/.config/skyr-rpc-explorer.env`
или в файле`.env` в рабочем каталоге обозревателя.
См. шаблон файла конфигурации [.env-sample](.env-sample) для изучения списка опций и подробностей форматирования `.env`.

Вы также можете передавать параметры в качестве аргументов CLI, например:

```bash
skyr-rpc-explorer --port 8080 --bitcoind-port 18443 --bitcoind-cookie ~/.bitcoin/regtest/.cookie
```

См. команду `skyr-rpc-explorer --help` для получения полного списка опций CLI.

## Запустить через Docker

1. `docker build -t skyr-rpc-explorer.`
2. `docker run -p 3002:3002 -it skyr-rpc-explorer`
